# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability in PrefMoE, please do
not open a public GitHub issue.

Instead, report it privately to:

zhangwenqi389@gmail.com

Please include the following when possible:

- a clear description of the issue
- the affected component or file path
- reproduction steps or a proof of concept
- the potential impact
- any suggested remediation

We will review reports as time permits and aim to acknowledge valid reports
promptly.
